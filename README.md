# Personal Portfolio

This is my personal portfolio website, built to showcase my experience as a full-stack web developer and highlight some of the work I’ve done.

The site itself is built with React, TypeScript, and Tailwind CSS, and is designed to be simple, responsive, and focused on demonstrating frontend development skills.

## 💻 Running Locally

If you'd like to run the project locally:

```bash
git clone https://github.com/YOUR_USERNAME/portfolio.git
cd portfolio
npm install
npm run dev
```
